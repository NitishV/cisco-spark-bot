My First Cisco Spark Bot

This is a sample Go web application deployed to Heroku. This site hosts a Cisco Spark bot which responds to certain events from Spark using webhooks.

Cisco Spark for Developers Source on GitHub

How this sample app works

  • This app was deployed to Heroku, either using Github integration.
  • When Heroku received the source code it used the go tool chain to compile the application along with any vendored dependencies and created a deployable slug.
  • The platform then spins up a dyno, a lightweight container that provides an isolated environment in which the slug can be mounted and executed.
  • You can scale your app, manage it, and deploy over 150 add-on services, from the Dashboard or CLI.
